Connect to Google Sheets

Paste your Apps Script Web App URL. Once connected, data syncs directly to your Sheet and persists forever.

Extensions → Apps Script → Deploy → New deployment in your Google Sheet
Setup instructions
  1. 1Download the KisnaMDF_GoogleAppsScript.js file
  2. 2Open a new Google Sheet → Extensions → Apps Script
  3. 3Paste the entire script and save (Ctrl+S)
  4. 4Click Deploy → New deployment → Type: Web App
  5. 5Set access to "Anyone" → Deploy → copy URL
  6. 6Paste the URL above and click Connect & sync
